# ガイド：session-start スキルの使い方（AIカンパニー）

## このスキルは何をするもの？

新しい開発セッションを始める際に、Antigravityが自動で以下を一括実行するスキルです。

- ROADMAPの現在地を確認・復元する
- 前回の学習メモを読み込む
- Julesの夜間作業成果を確認・同期する
- 今日の最初の5分アクションを定義する

## 使い方

新しいチャットを開いたら、最初にこの「魔法のプロンプト」をコピペするだけです。

```
AIカンパニー開発を再開します。docs/design/ 配下の設計書と
docs/learning/ の最新学習メモを読み込み、ROADMAPの現在地を確認してから、
前回の続きの作業を提案してください。
```

> ※ session-wrap-up の「ステップ6」で毎回このプロンプトが自動生成されます

## 出力されるもの

以下のフォーマットでセッション再開宣言が行われます：

```
### 🌅 セッション再開準備完了（AIカンパニー）

【現在の状況】
- ROADMAPの現在地: ノード○ 完了 → ノード○ 着手中
- Julesによる変更: ○○

【今日の最初のアクション（5分以内）】
- [ ] ROADMAPノード○: [具体的な作業]
```

## ROADMAPのノード順（優先ルール）

| ノード | 内容 | 優先順位 |
|---|---|---|
| A | DBスキーマ実装 | 最優先（未完なら必ずここから） |
| B | 4層マージロジック | Aの次 |
| C1 | S1 初回起動・APIキー設定 | Bの次 |
| C9 | S9 設定画面 | C1の次 |
| D2 | S2 プロジェクト一覧 | 並行 |
| D3 | S3 プロジェクト作成 | D2の次 |
| D4 | S4 チーム管理 | D3の次 |
| E5 | S5 メンバーエディタ | D4の次 |
| E6 | S6 1on1チャット | D4の次（並行） |
| F7A | 会議モード選択 | E5の次 |
| F7 | S7 会議モード | F7Aの次 |
| G8 | S8 議事録・サマリー | F7の次 |
| H | Markdownエクスポート | G8の次 |

## 注意点

- Julesのアーカイブ処理はしいたけさんが確認してから行います（自動削除はしません）
- `npm run dev` / `cargo tauri dev` の自動起動は行いません（確認後に提案します）
