# ガイド：session-wrap-up スキルの使い方（AIカンパニー）

## このスキルは何をするもの？

開発セッションを終了する際に、Antigravityが自動で以下を一括実行するスキルです。

- ROADMAPの進捗を更新する
- 学習メモを生成する（NotebookLM連携用）
- Jules向けの指示書を生成する
- Gitコミット用のコマンドを提示する

## 使い方

開発セッションを終えたいタイミングで、以下のように話しかけるだけです。

```
セッション終了します。
```

または

```
wrap-up お願いします。
```

## 出力されるもの

1. **今日やったことのサマリー**（ROADMAPのノード単位）
2. **Git一括コマンド**（コピー＆ペーストするだけ）
3. **Jules Web UI用のプロンプト**（短い1行テキスト）
4. **次回の魔法のプロンプト**（次のチャット開始時にコピペするだけ）

## 生成されるファイル

| ファイル | 用途 |
|---|---|
| `docs/learning/学習メモ_YYYYMMDD.md` | NotebookLMへのインポート用・自己学習記録 |
| `docs/jules-tasks/prompt_for_jules_YYYYMMDD.md` | Jules夜間作業への指示書 |

## AIカンパニー固有の注意点

- **3層構造・4層マージへの変更**はAntigravityが自動更新せず、しいたけさんに確認を求めます
- **Tauriのバックエンド（`src-tauri/`配下）**の変更はJulesへの依頼対象外です（Rustの変更は複雑なため）
- ROADMAPのノードが完了したときは、必ずwrap-upを実行してから次のノードに進んでください
