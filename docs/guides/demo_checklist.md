# デモ動作確認手順書

## AI Team Builder（AIカンパニー）結合テスト手順

> **目的**: Phase 1 の全画面（S1〜S9）が正しく連携することを確認する。
> **所要時間**: 約10分
> **前提**: `npm run tauri dev` でアプリが起動していること

---

## チェックリスト

### 1. 初回起動（S1）
- [ ] アプリを起動するとAPIキー設定画面が表示される
- [ ] OpenAI / Anthropic / Gemini のいずれかのキーを入力して保存できる
- [ ] 「接続テスト」ボタンで正しく応答が返る
- [ ] 保存後、プロジェクト一覧画面（S2）に遷移する

### 2. プロジェクト一覧（S2）
- [ ] 空の状態で「まだプロジェクトがありません」と表示される
- [ ] 「新しいプロジェクトを作成」ボタンが表示される
- [ ] 設定アイコン（⚙️）から設定画面（S9）に遷移できる

### 3. プロジェクト作成（S3）
- [ ] プロジェクト名・目的・価値観を入力できる
- [ ] 部署プリセット（経営・戦略 / エンジニアリング / 法務 / マーケティング / 思考スタイル）が選択できる
- [ ] 各部署の初期メンバーが自動生成される
- [ ] 作成後、チーム管理画面（S4）に遷移する

### 4. チーム管理（S4）
- [ ] 部署タブが横並びで表示される
- [ ] 部署を切り替えると所属メンバーが変わる
- [ ] メンバーカードにアバター・役割・モデルが表示される
- [ ] 「話す」ボタンで1on1チャット（S6）に遷移する
- [ ] 「会議を開始」ボタンで会議モード選択モーダル（F7A）が表示される

### 5. メンバーエディタ（S5）
- [ ] メンバーカードをクリックするとエディタが開く
- [ ] 継承元パネル（プロジェクト価値観・部署性質）が表示される
- [ ] 成長日誌（自動学習履歴）が表示される
- [ ] 役割・人格プロンプト・AIモデルを編集できる

### 6. 1on1チャット（S6）
- [ ] メッセージを入力して送信できる
- [ ] AIメンバーから返答が返ってくる
- [ ] チャット履歴が保持される（画面遷移後も残る）
- [ ] 「チャット履歴をクリア」で削除できる

### 7. 会議モード（S7）
- [ ] 探索モードと収束モードの両方が選択できる
- [ ] 会議が開始され、メンバーがラウンドロビンで発言する
- [ ] 発言中メンバーがハイライト表示される
- [ ] 一時停止ボタンで発言が止まる
- [ ] 再開ボタンで続きから再開する
- [ ] 割り込みボタンが機能する（連続3回まで）
- [ ] 全メンバーが発言し終わると「サマリーを生成」ボタンが表示される

### 8. 議事録・サマリー（S8）
- [ ] 会議の論点・各メンバーの見解が表示される
- [ ] 決定事項を入力できる
- [ ] 「Markdownで保存」ボタンでファイル保存ダイアログが開く
- [ ] 保存したファイルが正しいMarkdown形式になっている
- [ ] 「議事録を保存」で決定事項がDBに保存される

### 9. 設定画面（S9）
- [ ] 左メニューと右コンテンツの2ペイン構成になっている
- [ ] 「AI・API設定」タブでAPIキーの追加・削除ができる
- [ ] 「コアプロフィール」タブで自由記述のプロフィールを編集できる
- [ ] コアプロフィールの内容が会議のシステムプロンプトに反映される

### 10. クロス画面テスト
- [ ] S2 → S3 → S4 → S5 → S4 → S6 → S4 → S7 → S8 → S4 の一連の遷移がエラーなく行える
- [ ] ブラウザバック相当の操作（「戻る」ボタン）が正しく機能する
- [ ] 未設定のAPIキーに対して適切なエラーメッセージが表示される

---

## 既知の制限・注意点

- **APIキー**: テストには実際のAPIキーが必要。テスト用のキーを事前に準備すること
- **会議の所要時間**: メンバー数×3ターン×API応答時間。3メンバーで約2〜3分
- **Tauri v2 パーミッション**: `src-tauri/capabilities/default.json` の権限設定が不足していると、ファイル保存・ダイアログが動作しない。`fs:allow-write-text-file` 等の権限が含まれていることを確認すること
- **初回ビルド**: `npm run tauri dev` の初回起動はRustのコンパイルが入るため、数分かかる
