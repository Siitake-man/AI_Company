# Jules 学習メモ (2026-07-14)

## 今回学んだCSSテクニック・ポイント

1. **立体感の表現 (Box Shadow)**
   - `box-shadow: 0px 4px 0px var(--color-border-inner)` のように、ぼかし（blur）を `0px` に設定し、Y軸方向にオフセットをつけることで、フラットデザインではなく「物理的な厚み（木材や看板の厚み）」を表現できる。
   - `inset` シャドウと通常のシャドウを組み合わせることで（例: `.panel-paper`）、紙が少し浮いているような効果と、内側に微細なハイライトを入れるテクニックを学んだ。

2. **押し込みアニメーション**
   - `:active` 疑似クラスで `transform: translateY(2px)` などを用いて要素を沈み込ませると同時に、`box-shadow` を `none` または小さくすることで、物理的なボタンが押された感覚を表現できる。

3. **TailwindとカスタムCSSの使い分け**
   - Tailwindのユーティリティクラス（`flex`, `p-4`, `gap-2` など）はレイアウト構築に便利だが、本プロジェクトのように明確なテーマ（箱庭風・レトロ）がある場合、コンポーネントの質感に関わる詳細なスタイル（特殊なボーダー、シャドウの多重がけ）は、`index.css` に `.sidebar-item` のようなカスタムクラスとして定義し、CSS変数を適用するアプローチが保守性の観点から優れていることを再確認した。

4. **スクロール制御**
   - 最外枠を `h-screen overflow-hidden` で固定し、内部のコンテンツエリアに `flex-1 overflow-y-auto` を適用することで、ウィンドウ全体のスクロールを防ぎつつ、必要な領域だけをスクロール可能にするレイアウトパターンを活用した。
