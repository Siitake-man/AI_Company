**[現在の症状]**
1. アプリ全体（`main`タグ等）がウィンドウサイズを超えてしまい、下部の「新しいプロジェクト」ボタン等が見切れることがある。
2. メインコンテンツ領域（メンバー一覧が表示されている右側のエリア等）で、メンバー数が増えても縦スクロール（`overflow-y-auto`）が効かず、画面外に溢れてしまう。

- 画面全体の高さはOSのウィンドウ高さにピッタリ固定する（`100vh` または `height: 100%` と `overflow: hidden` の組み合わせ）。
- ヘッダーと左サイドバーはスクロールさせず固定表示する。
- 右側の「メインコンテンツ表示エリア」**のみ**、コンテンツが溢れた場合に縦スクロールバーが表示されてスクロールできるようにする。
